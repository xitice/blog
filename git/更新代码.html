# pagecook项目如何更新代码

  1. 关闭当前分支所有文件窗口 (feature-sjc)
  2. git checkout develop 切换到develop分支
  3. git pull 拉取最新代码
  4. git checkout feature-sjc 切换到feature-sjc分支
  5. git merge develop 将develop最新代码合并到当前分支

# 如何回滚版本

  1. git log 显示提交版本信息

  1. git reset --hard commit的前几位字符

  2. 就会回滚到指定的版本
